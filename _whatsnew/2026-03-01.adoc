=== NetApp Console agent status supported for FSx for ONTAP file systems

NetApp Console now supports displaying the Console agent status for FSx for ONTAP file systems from within Workloads. The status indicates whether the NetApp Console agent is running and able to communicate with the FSx for ONTAP file system. This information can help you troubleshoot connectivity issues and ensure that your FSx for ONTAP file system is properly integrated with the NetApp Console.
